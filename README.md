# resume

AI 자동화 엔지니어 이력서 — Jekyll · GitHub Pages

- 테마: [sproogen/modern-resume-theme](https://github.com/sproogen/modern-resume-theme) (remote theme)
- 배포: https://min-omniai.github.io/resume/

## 수정 방법

내용은 전부 `_config.yml` 한 파일에서 관리합니다 (소개·프로젝트·경력·학력·스킬).
이미지는 `images/` (프로필·파비콘).

## 로컬 미리보기

```bash
bundle install
bundle exec jekyll serve
```

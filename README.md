# resume

AI 자동화 엔지니어 이력서 — Jekyll 기반 GitHub Pages 사이트.

- 테마: [sproogen/resume-theme](https://github.com/sproogen/resume-theme) (remote theme)
- 구조 레퍼런스: [richrookie/resume](https://github.com/richrookie/resume)
- 내용 원본: ops-smart04 `10_Review/6단계_AI엔지니어_지향판.md`

## 로컬 미리보기

```bash
bundle install
bundle exec jekyll serve
```

## 수정 방법

- 인적 사항·소개: `_config.yml`
- 경력: `_data/experience.yml`
- 프로젝트: `_data/projects.yml`
- 스킬: `_data/skill.yml`
- 학력·자격: `_data/education.yml`
- 오픈소스: `_data/opensource.yml`
